# MIMO Application Text

## Direct paste version

我使用 Agent/AI 驱动构建了一个面向遥感视频超分研究的多 Agent 研究助手原型 `MADNet-Research-Agent`。它解决的核心痛点是：遥感 VSR 相关论文、退化建模、数据集和评测指标高度分散，人工检索和手工归纳效率低且容易遗漏证据，导致从“找论文、比方法、定实验、写结论”到最终产出之间存在很长且不稳定的研究链路。这个项目将流程拆成 `Planner / Retriever / Analyst / Critic / Writer` 五类 Agent，形成一条可复核的长链推理逻辑：先把研究问题分解为可执行子目标，再针对方法、退化类型、数据集、指标等维度做证据召回与结构化分析，随后由 Critic Agent 检查证据缺口、潜在偏差和不充分结论，最后由 Writer Agent 输出报告和推理轨迹。相较于一次性问答，这种多 Agent 协作方式更适合研究场景，能够把复杂问题拆解、证据化和审校化，方便复现、核查和持续迭代。

## Shorter version

我构建了一个面向遥感视频超分研究的多 Agent 助手原型，用来解决论文检索分散、实验设定难比较、证据归纳低效且容易遗漏的问题。系统采用 `Planner-Retriever-Analyst-Critic-Writer` 的多 Agent 协作链路，包含显式的长链推理与中间状态传递：先分解研究问题，再检索和结构化分析证据，最后通过批判性复核后生成可核查结论。这让“找论文-比方法-提实验-写结论”的研究流程从人工串行处理转成可复现、可审查、可迭代的 Agent 工作流。
